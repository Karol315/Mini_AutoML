## Struktura plików:

```
Mini_AutoML/
├─ data/                # Dane treningowe i testowe
│  ├─ train.csv
│  └─ test.csv
├─ src/                 # Kody źródłowe
│  ├─ preprocessing.py
│  ├─ model.py
│  └─ utils.py

```

## Opis projektu:

Mini_AutoML to projekt mający na celu automatyzację procesu tworzenia modeli ML z minimalną ingerencją użytkownika. Projekt zawiera przetwarzanie danych, trenowanie modeli oraz narzędzia pomocnicze do oceny wyników.
